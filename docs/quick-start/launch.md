# 启动

## 启动命令

```shell
    zan-proxy
```

启动ZanProxy会自动打开ZanProxy的管理页。

<img src="https://img.yzcdn.cn/public_files/2018/04/17/7dbc1ee4b763d7f8f7b21310200ce238.png">

## 启动参数

### 指定代理端口

```shell
    zan-proxy -p 8002
```

### 指定管理服务器端口

```shell
    zan-proxy -u 40002
```

### 查看所有命令

```shell
    zan-proxy -h
```
