使用方式

使用方式

创建新的工作区

点击左下角的新建按钮，即可创建一个新的工作区预设

配置工作区内容

新建或点击左侧任意工作区后，可以在右侧的工作区编辑区域修改当前工作区的配置。

修改完毕后，点击右上角的保存按钮保存工作区配置

切换工作区

点击左侧列表中对应工作区的开关即可切换到对应的工作区

或者点击菜单栏右侧的 zan-proxy-mac 图标，在弹出的菜单中切换工作区环境

导出和导入工作区

点击右上角的导出按钮可以导出当前工作区的配置，默认将导出的配置文件存放在 ~/Download 目录下

需要导入其他人的配置时，可以点击左下角的导入按钮，选择对应的 json 工作区配置文件进行导入

复制工作区

点击右上角的复制按钮，即可复制一份与当前工作区配置相同的工作区预设

删除工作区

当不再需要某个工作区时，点击右上角的删除按钮，即可删除该工作区
