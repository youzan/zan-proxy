chrome代理配置

chrome代理配置

  • 安装chrome代理设置插件 SwitchyOmega

  • 进入SwitchyOmega的配置页，新建情景模式。

  • 设置情景模式的代理协议为HTTP，代理服务器为127.0.0.1，端口为ZanProxy的代理端口，默认为8001

  • 当需要使用ZanProxy代理请求时，将SwitchyOmega选择为你为ZanProxy设置的情景模式。
